Vim commands

  • -v: Get Vim Version
  • : Colon : Go into command mode
  • q!Quit Vim without saving
  • i: Insert mode
  • Esc: Exit insert mode
  • :w: Save file
  • :q: Quit Vim
  • :wq: Save and quit
  • dd: Delete current line
  • yy: Copy (yank) current line
  • p: Paste copied line
  • u: Undo last action
  • /text: Search for "text"
